The Admin pages are used to administer base data and settings for the Niord application. It has been divided into the following sections:

Charts
Configure the nautical charts.
Areas
Manage the list of areas used for grouping NW and NM messages.
Categories
Manage the list of categories used for grouping NW and NM messages.
Integration
Handle integration with the legacy systems.
Batch Jobs
Monitor and manage batch jobs.
Settings
Manage the system settings.
