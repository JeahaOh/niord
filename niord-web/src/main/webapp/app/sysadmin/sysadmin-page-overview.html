The System Admin pages are used to administer system data and settings for the Niord application. It has been divided into the following sections:

Message Series
Manage the message series used for message series identifier.
Domains
Manage the domains.
Schedules
Manage the list of firing schedules.
Dictionaries
Manage the dictionaries.
Templates
Manage the Freemarker templates used for message templates, print reports and mails.
Print Reports
Manage the PDF print reports.
Scheduled Mails
Search all sent e-mails.
Integration
Handle integration with the legacy systems.
Settings
Manage the system settings.
Batch Jobs
Monitor and manage batch jobs.
